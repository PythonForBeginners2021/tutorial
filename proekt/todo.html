todo

Описание проекта

Написать консольное приложениe для ведения списка задач - todo.

Приложение должно сохранять задачи в файле todo.txt в той же папке, где находится исполняемый модуль приложения.

Приложение должно поддерживать следующие команды:

  • todo.py - показать незавершенные задачи

  • todo.py all - показать все задачи (включая завершенные)

  • todo.py add <task description> [due:<date>] - добавить задачу с описание <task description> (и дополнительно можно добавить срок выполнения <date>)

  • todo.py remove <n> - удалить задачу с номером <n>

  • todo.py edit <n> <new task description> - редактировать задачу с номером <n>: установить новое описание <new task description>

  • todo.py due <n> <date> - установить срок выполнения <date> для задачи <n>

  • todo.py done <n> - пометить задачу как выполненную

  • todo.py undo <n> - снять отметку о выполнении

  • todo.py search <sting> - поиск задач по тексту <string>

Формат файла todo.txt:

  1. Поле завершенности: "x" (выставляется автоматически после выполнения команды done)

  2. Дата создания задачи: "2019-11-23" (автоматически при создании)

  3. Описание задачи

  4. Срок задачи: "due:2019-12-23" (не обязательное)

Пример файла todo.txt:

todo.txt
2019-11-22 Придумать финальный проект due:2019-11-23
2019-11-22 Купить молоко due:2019-11-25
2019-11-22 Купить обои due:2019-11-25
2019-11-22 Поклеить обои due:2019-12-01
x 2019-11-22 Дописать лекцию

Примеры использования:

Добавление задачи:

$ python3 todo.py add Дописать лекцию
​
TODO:
----------------------------------------------
1: 2019-11-22 Дописать лекцию - ДОБАВЛЕНО

Примечание: начало и конец строки "1: " и " - ДОБАВЛЕНО" не хранятся в файле, а добавляются динамически при выводе в консоль

Добавление задачи со сроком выполнения:

$ python3 todo.py add Придумать проект due:2019-11-23
​
TODO:
----------------------------------------------
1: 2019-11-22 Дописать лекцию
2: 2019-11-22 Придумать проект due:2019-11-23 - ДОБАВЛЕНО

Добавление срока выполнения к задаче 1:

$ python3 todo.py due 1 2019-11-23
​
TODO:
----------------------------------------------
1: 2019-11-22 Дописать лекцию due:2019-11-23 - ИЗМЕНЕН СРОК
2: 2019-11-22 Придумать проект due:2019-11-23

Вывод списка задач:

$ python3 todo.py
​
TODO:
----------------------------------------------
1: 2019-11-22 Дописать лекцию due:2019-11-23
2: 2019-11-22 Придумать проект due:2019-11-23

Завершение задачи:

$ python3 todo.py done 2
​
TODO:
----------------------------------------------
1: 2019-11-22 Дописать лекцию due:2019-11-23
2: x 2019-11-22 Придумать проект due:2019-11-23 - ЗАВЕРШЕНО

Примечание: завершенная задача помечается латинской буквой x в начале строки (и в консоли, и в файле), и перемещается в конец (и в файле, и при выводе в консоль)

Вывод списка (завершенная задача не выводится):

$ python3 todo.py
​
TODO:
----------------------------------------------
1: 2019-11-22 Дописать лекцию due:2019-11-23

Вывод списка всех задач (включая завершенные):

$ python3 todo.py all
​
TODO:
----------------------------------------------
1: 2019-11-22 Дописать лекцию due:2019-11-23
2: x 2019-11-22 Придумать проект due:2019-11-23

Пометить задачу как незавершенную:

$ python3 todo.py undo 2
​
TODO:
----------------------------------------------
1: 2019-11-22 Дописать лекцию due:2019-11-23
2: 2019-11-22 Придумать проект due:2019-11-23 - ВОССТАНОВЛЕНО

Удалить задачу 1 (и отменить процесс удаления):

$ python3 todo.py remove 1
1: 2019-11-22 Дописать лекцию due:2019-11-23
Хотите удалить задачу 1? (y/n) n
ЗАДАЧА НЕ УДАЛЕНА
​
TODO:
----------------------------------------------
1: 2019-11-22 Дописать лекцию due:2019-11-23
2: 2019-11-22 Придумать проект due:2019-11-23

Удалить задачу 1 (и подтвердить удаление):

$ python3 todo.py remove 1
1: 2019-11-22 Дописать лекцию due:2019-11-23
Хотите удалить задачу 1? (y/n) n
ЗАДАЧА УДАЛЕНА
​
TODO:
----------------------------------------------
1: 2019-11-22 Придумать проект due:2019-11-23

Редактировать задачу 1:

$ python3 todo.py edit 1 Придумать финальный проект
​
TODO:
----------------------------------------------
1: 2019-11-22 Придумать финальный проект due:2019-11-23 - ИЗМЕНЕНО

(Добавим еще пару задач, чтобы заполнить список:)

$ python3 todo.py add Дописать лекцию
​
TODO:
----------------------------------------------
1: 2019-11-22 Придумать финальный проект due:2019-11-23
2: 2019-11-22 Дописать лекцию - ДОБАВЛЕНО
​
$ python3 todo.py add Купить молоко due:2019-11-25
​
TODO:
----------------------------------------------
1: 2019-11-22 Придумать финальный проект due:2019-11-23
2: 2019-11-22 Дописать лекцию
3: 2019-11-22 Купить молоко due:2019-11-25 - ДОБАВЛЕНО
​
$ python3 todo.py add Купить обои due:2019-11-25
​
TODO:
----------------------------------------------
1: 2019-11-22 Придумать финальный проект due:2019-11-23
2: 2019-11-22 Дописать лекцию
3: 2019-11-22 Купить молоко due:2019-11-25
4: 2019-11-22 Купить обои due:2019-11-25 - ДОБАВЛЕНО
​
$ python3 todo.py add Поклеить обои due:2019-12-01
​
TODO:
----------------------------------------------
1: 2019-11-22 Придумать финальный проект due:2019-11-23
2: 2019-11-22 Дописать лекцию
3: 2019-11-22 Купить молоко due:2019-11-25
4: 2019-11-22 Купить обои due:2019-11-25
5: 2019-11-22 Поклеить обои due:2019-12-01 - ДОБАВЛЕНО

Поиск по задачам:

$ python3 todo.py search обои
​
TODO: <обои>
----------------------------------------------
1: 2019-11-22 Купить обои due:2019-11-25
2: 2019-11-22 Поклеить обои due:2019-12-01
​
​
$ python3 todo.py search купить
​
TODO: <купить>
----------------------------------------------
1: 2019-11-22 Купить молоко due:2019-11-25
2: 2019-11-22 Купить обои due:2019-11-25
​
$ python3 todo.py search due:2019-11-25
​
TODO: <due:2019-11-25>
----------------------------------------------
1: 2019-11-22 Купить молоко due:2019-11-25
2: 2019-11-22 Купить обои due:2019-11-25

Дополнительное задание:

Реализовать поиск по датам:

  • created between <date> <date> - поиск по дате создания между указанными датами

  • created before <date> - поиск по дате создания до указанной даты

  • created after <date> - поиск по дате создания после указанной даты

  • due between <date> <date> - поиск по дате выполнения между указанными датами

  • due before <date> - поиск по дате выполнения до указанной даты

  • due after <date> - поиск по дате выполнения после указанной даты
