Практика. Запуск кода на Python из файла

В рамках этого практикума нашей задачей будет научиться запускать программу на Python. Предварительно мы напишем и сохраним ее в файле с расширением *.py. Сейчас нам не так важно, что это будет за программа, поэтому ограничимся выводом на экран фразы “Hello World!”.

Итак, вот что тебе нужно сделать:

  • Для начала создай папку для заданий курса. Можешь назвать ее как угодно, но желательно использовать латиницу и избегать пробелов. В нашем примере мы назовем эту папку practice.

  • Открой папку practice в выбранном тобой редакторе кода и создай новый файл hello.py.

  • Напечатай в файле hello.py строку:

hello.py
print(“Hello World!”)

  • Затем открой окно командной строки/консоли/терминала (уточни, как это можно сделать в твоей операционной системе) и найди в ней папку practice. В этом тебе могут помочь следующие команды:

Windows

Mac / Linux

Для чего нужна команда?

cd

cd

Перейти в нужную папку (директорию). Пример:

cd practice

Чтобы вернуться в предыдущую директорию:

cd ..

dir

ls

Посмотреть список файлов в директории

  • Как только ты оказалась в папке practice и проверила, что файл hello.py находится в ней, тебе остается только выпонить команду python3.6 hello.py

Обрати внимание, что в примере мы используем версию Python3.6. Если на твоем компьютере установлена другая версия Python3, то для запуска файла с кодом на Python тебе нужно использовать ее.

Результат выполнения должен быть следующим:

Поздравляю! Ты только запустила программу, написанную на Python!
