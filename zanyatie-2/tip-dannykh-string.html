Тип данных str

Несмотря на то, что компьютер понимает только язык цифр, у нас есть возможность работать еще и с текстом. Для этого во многих языках программирования есть отдельный тип данных, который называется str (string – строка). Это очень важный и часто используемый тип данных, так как существует огромное количество задач, связанных с обработкой текста.

Для того, чтобы создать значение типа str, вам достаточно напечатать любую последовательность символов и обернуть ее в двойные ("строка") или одинарные ('строка') кавычки. 

>>> type("python")
<class 'str'>
>>> type('24352.5354')
<class 'str'>
>>> type("@*$#&@)$#")
<class 'str'>

Строка, которую ты хочешь создать может быть абсолютно любой длины, единственное ограничение – это память твоей системы. Ты можешь создать даже пустую строку! 

>>> ""
''

Если ты открываешь строку двойными кавычками, то и закрывать ее нужно двойными. Никогда не смешивай двойные и одинарные кавычки для обозначения строки, иначе интерпретатор Python не поймет, что строка закончилась, и выдаст ошибку. 

>>> 'test"
  File "<stdin>", line 1
    'test"
         ^
SyntaxError: EOL while scanning string literal

Преобразовать в строку можно любой тип данных с помощью функции str() 

>>> str(14324)
'14324'
>>> str(True)
'True'
>>> str(0.0435)
'0.0435'
>>>

Строки также можно перевести и в другие типы данных. Значением True становится любая непустая строка, а False – пустая. 

>>> bool('test')
True
>>> bool('')
False

В int и float можно преобразовать только строку, которая состоит из чисел. Если str будет содержать хотя бы один другой символ – интерпретатор выдаст нам ошибку. 

>>> int('5')
5
>>> float('5.243')
5.243
>>> int('test')
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ValueError: invalid literal for int() with base 10: 'test'
>>> float('test')
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ValueError: could not convert string to float: 'test'

Значение типа str можно присваивать переменной, как и любой другой тип данных. Сделать это очень просто: 

>>> my_string = "Hello World"
>>> my_string
'Hello World'

Иногда тебе нужно будет напечатать кавычки внутри кавычек. Сделать это можно двумя способами:

  1. используй разные типы кавычек для создания строки и внутри этой самой строки:

 >>> 'Эта строка такая "полезная"'
 'Эта строка такая "полезная"'

  1. экранируй кавычки с помощью символа \ (обратный слэш) - тогда интерпретатор будет знать, что их следует воспринимать как символы, а не считать началом или окончанием строки:

 >>> "Я прочла \"Войну и мир\" и мне не понравилось"
 'Я прочла "Войну и мир" и мне не понравилось'

Для того, чтобы напечатать текст, разбитый на несколько строк, у нас тоже есть два способа:

  • Мы можем использовать экранированную последовательность \n для обозначения места, с которого мы хотим начать новую строку:

 >>> print("Я помню чудное мгновенье\nПередо мной явилась ты")
 Я помню чудное мгновенье
 Передо мной явилась ты

  • Обрати внимание, что \n работает только когда ты выводишь текст на экран при помощи функции print(). Если использовать его в интерпретаторе Python, это не даст нужный нам эффект:

 >>> "Я помню чудное мгновенье\nПередо мной явилась ты"
 'Я помню чудное мгновенье\nПередо мной явилась ты'

  • Для многострочного текста мы также можем использовать тройные кавычки:

 >>> print("""Я помню чудное мгновенье
 ... передо мной явилась ты""")
 Я помню чудное мгновенье
 передо мной явилась ты

  • Однако при попытке использовать его в интерпетаторе, ты можешь обнаружить все тот же \n:

>>> """Я помню чудное мгновенье
 ... Передо мной явилась ты"""
 'Я помню чудное мгновенье\nПередо мной явилась ты'
