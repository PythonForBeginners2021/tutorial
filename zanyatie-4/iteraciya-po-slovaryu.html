Итерация по словарю

Самый простой способ итерировать по словарю – по его ключам. Вспомним про наш словарь friends. Чтобы поочередно добраться до каждого из его значений, нам понадобится цикл for, как и в случае с остальными составными типами данных.

birthdays.py
friends = {
    "Света": "02.02.1990",
    "Маша": "01.01.1991",
    "Карина": "03.03.1993"
}
for name in friends:
    print(name)

Но что если мы хотим использовать не только ключи, но и значения словаря? Как ты помнишь, значение мы можем получить, обращаясь к словарю по ключу, который он содержит. То же самое мы можем сделать и в процессе итерации.

birthdays.py
friends = {
    "Света": "02.02.1990",
    "Маша": "01.01.1991",
    "Карина": "03.03.1993"
}
for name in friends:
    print(name, friends[name])

Помимо этого, мы можем использовать методы .keys() и .values(), чтобы узнать, какие в нашем словаре есть ключи и значения соответственно.

>>> friends.keys()
dict_keys(['Маша', 'Карина'])
>>> friends.values()
dict_values([{'Крапивина': '09.09.2000', 'Жукова': '01.01.1991'}, '05.05.1987'])

И если нам захочется итерировать только по ключам (или только по значениям) словаря, мы можем обратиться к ним.

birthdays.py
friends = {
    "Света": "02.02.1990",
    "Маша": "01.01.1991",
    "Карина": "03.03.1993"
}
for name in friends.keys():
    print(name)
for date in friends.values():
    print(date)

Однако если ты используешь словари, тебе чаще всего будет нужно итерировать одновременно по ключам и значениям словаря. Ты все еще можешь использовать для этого первый способ, обращаясь к словарю по ключу, чтобы получить значение. Однако в Python есть более простой и полезный способ итерации по словарю, который позволит тебе создать отдельный итератор для значений словаря.

Для этого метода тебе понадобится встроенный метод .items(), который создает список с кортежами. Первым элементом кортежа всегда является ключ, а вторым – значение.

>>> friends.items()
dict_items([('Света', '02.02.1990'), ('Маша', '01.01.1991'), ('Карина', '03.03.1993')])

Таким образом, мы можем использовать два итератора – для ключа и значения.

birthdays.py
friends = {
    "Света": "02.02.1990",
    "Маша": "01.01.1991",
    "Карина": "03.03.1993"
}
for name, date in friends.items():
    print(name, date)

Помнишь задачу, которую мы поставили перед тобой в самом начале раздела про тип данных dict? Давай попробуем решить ее, используя все полученные знания про словари.

birthdays.py
user_name = input("Введите имя: ")
friends = {
    "Света": "02.02.1990",
    "Маша": "01.01.1991",
    "Карина": "03.03.1993"
}
for name, date in friends.items():
    if name == user_name:
        print(f"День рождения: {date}")
        break

Обрати внимание, что переменная-итератор (в нашем случае, name) должна быть уникальной, Поэтому пользовательский ввод мы обозначили как user_name – чтобы названия переменных не совпадали.

Нам нужно выводить дату рождения только в случае, если имя, которое ввел пользователь, совпадет с именем в словаре. Если имя совпало – мы печатаем дату рождения, а также прерываем выполнение цикла, чтобы не производить лишние действия (ведь наш словарь может быть очень большим).
