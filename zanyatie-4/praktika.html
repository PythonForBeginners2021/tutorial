Акроним

Акроним – или аббревиатура – это слово, состоящее из первых букв некоей фразы. Например, ООО – это Общество с Ограниченной Ответственностью, YMCA – Young Men’s Christian Association и т.д. Твоя задача – написать программу, которая будет превращать вводимую пользователем фразу в акроним.

Для решения этой задачи тебе понадобится еще один очень полезный строковый метод, который мы не рассматривали – str.join(). Он склеивает объекты списка (или другой последовательности) в строку.

>>> numbers = ["1", "2", "3", "4", "5"]
>>> "".join(numbers)
'12345'

Метод применяется к строке, которую мы хотим использовать в качестве разделителя – в данном случае это пустая строка "", поэтому элементы из списка numbers склеились в единую строку.

Но мы также можем использовать различные разделители. Главное условие – разделитель должен быть строкой.

>>> " ".join(numbers)
'1 2 3 4 5'
>>> "-".join(numbers)
'1-2-3-4-5'
>>> " и ".join(numbers)
'1 и 2 и 3 и 4 и 5'

Обрати внимание, что все объекты списка должны быть типа str, иначе использование метода приведет к ошибке.

>>> numbers = [1, 2, 3, 4, 5]
>>> "".join(numbers)
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: sequence item 0: expected str instance, int found

Магазин

Наверняка ты знаешь, что у каждого магазина есть некая база, по которой продавцы могут пробить товар и узнать, есть ли он в наличии. Сейчас тебе нужно будет написать такую систему для небольшого магазина!

Пользователь должен иметь возможносить вбить категорию товара и продукт. Если указана несуществующая категория, то программа должна выводить список доступных категорий и предложить сделать ввод вновь.

Если введена существующая категория, то должно появится поле для ввода товара. Программа выводит соответствующее сообщение в зависимости от того, есть ли товар на складе и завершает работу.

В программе должна храниться информация о том, что:

  • есть 3 категории товаров: фрукты, овощи и сладости

  • из фруктов на складе есть яблоки, бананы, апельсины и манго

  • из овощей есть помидоры, картошка, тыква и огурцы

  • из сладостей – конфеты, шоколад, пирог, печенье

Твоя задача, помимо всего прочего, организовать хранение данных на складе так, чтобы удобно было искать. Если у тебя возникнут трудности – загляни в подсказки!

С хранением данных мы разберемся на следующем занятии, а сейчас давай попробуем реализовать основные задачи программы: добавление и поиск по телефонной книге.

Давай по порядку разберемся с пунктами, которые тебе необходимо реализовать. Обрати внимание, что мы укажем здесь только основные моменты, часть требований ты должна определить и реализовать самостоятельно, глядя на пример работы программы. Если у тебя возникнут затруднения – можешь подсмотреть в раздел с подсказками :)
