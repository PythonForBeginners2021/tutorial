Импорт кода из исполняемого файла

На прошлом занятии мы говорили о том, что программа имеет свойство разрастаться. Поэтому для удобства чтения и облегчения дальнейшей работы над программой мы выносим некоторые блоки кода в отдельные функции, выполняющие конкретные задачи. Как же сделать так, чтобы мы могли использовать одну и ту же функцию (или переменную) в разных наших программах, не дублируя при этом код?

У нас есть программа, помогающая выбрать тип верхней одежды для одного из 4 времен года.

weather.py
seasons = {
    "зима": "пуховик",
    "весна": "пальто",
    "лето": "ветровка",
    "осень": "куртка",
}
​
today = input("Какое сейчас время года? ")
for season, cloth in seasons.items():
    if season == today:
        print(cloth)

А еще есть программа, которая помогает выбрать блюдо дня в ресторане.

restraunt.py
dishes = {
    "понедельник": "суп",
    "вторник": "стейк",
    "среда": "паста",
    "четверг": "отбивная",
    "пятница": "фрикадельки",
}
​
today = input("Какой сегодня день? ")
for day, dish in dishes.items():
    if day == today:
        print(dish)

Как ты можешь заметить, обе программы делают примерно одно и то же - ищут в словаре ключ, соответствующий пользовательскому вводу, и выводят его значение. Если мы вынесем одинаковый код в отдельную функцию, то получим две абсолютно идентичные функции print_value().

weather.py
def print_value(item, dictionary):
    for key, value in dictionary.items():
        if key == item:
            print(value)
​
seasons = {
    "зима": "пуховик",
    "весна": "пальто",
    "лето": "ветровка",
    "осень": "куртка",
}
​
today = input("Какое сейчас время года? ")
print_value(today, seasons)
restraunt.py
def print_value(item, dictionary):
    for key, value in dictionary.items():
        if key == item:
            print(value)
​
dishes = {
    "понедельник": "суп",
    "вторник": "стейк",
    "среда": "паста",
    "четверг": "отбивная",
    "пятница": "фрикадельки",
}
​
today = input("Какой сегодня день? ")
print_value(today, dishes)

Функция print_value() в обеих файлах дублируется. Однако Python позволяет нам получить доступ к коду другого файла, импортировав его в текущий файл с помощью ключевого слова import и названия файла, содержащего нужный нам код. На самом деле импорт можно произвести множеством разных способов, о которых мы поговорим чуть позже, но пока будем использовать этот вариант. Благодаря этой возможности, мы можем избавиться от дублирующего кода!

Итак, мы хотим оставить функцию print_value() в файле weather.py и импортировать ее в файл restraunt.py. Содержимое файла weather.py остается неизменным:

weather.py
def print_value(item, dictionary):
    for key, value in dictionary.items():
        if key == item:
            print(value)
​
seasons = {
    "зима": "пуховик",
    "весна": "пальто",
    "лето": "ветровка",
    "осень": "куртка",
}
​
today = input("Какое сейчас время года? ")
print_value(today, seasons)

А из файла restraunt.py мы убираем функцию print_value() и выполняем импорт файла weather.py - в самой первой строчке пишем import weather. Обрати внимание, что теперь для вызова функции print_value() мы сначала должны указать название импортируемого файла и только после этого, через точку, название функции - weather.print_value().

restraunt.py
import weather
​
dishes = {
    "понедельник": "суп",
    "вторник": "стейк",
    "среда": "паста",
    "четверг": "отбивная",
    "пятница": "фрикадельки",
}
​
today = input("Какой сегодня день? ")
weather.print_value(today, dishes)

Теперь нам не нужно писать одну и ту же функцию в разных файлах, мы просто воспользовались подходящей из другого файла! Давай попробуем запустить обе программы и проверить, что они по-прежнему работают.

Программа в файле weather.py работает как и раньше:

Однако при запуске файла restraunt.py начинает происходить что-то странное:

По какой-то причине код из файла weather.py начал выполняться, хотя мы хотели всего лишь использовать одну функцию! Дело в том, что файл weather.py является исполняемым, т.е. в нем происходит вызов функций и произведение неких операций. Речь про эти строчки:

today = input("Какое сейчас время года? ")
print_value(today, seasons)

Поэтому при импорте его содержимого помимо неисполняемого кода (функции print_value() и словарь seasons) в файл restraunt.py попадает еще и исполняемый (строки с вызовом функций). Мы можем убрать эти строчки из файла weather.py, чтобы при выполнении файла restraunt.py не происходило ненужных действий, однако после этого мы не сможем воспользоваться нашей программой в файле weather.py. Что же делать?

Когда файл импортируется, интерпретатор Python присваивает имя импортируемого файла специальной переменной __name__. Однако, если файл исполняется как скрипт (то есть, когда ты запускаешь его в консоли), переменной __name__ присваивается строка '__main__'. Зная об этом, мы теперь можем определить, какой случай в данный момент имеет место, с помощью условия if __name__ == '__main__', в тело которого мы поместим строчки исполняемого кода. Если это условие выполняется, значит файл запущен из консоли, и скрипт должен выполнить код в теле этого условия. В противном случае - когда мы производим импорт этого файла в другой - условие не выполняется, но при этом мы получаем доступ к функции print_value() и словарю seasons.

weather.py
def print_value(item, dictionary):
    for key, value in dictionary.items():
        if key == item:
            print(value)
​
seasons = {
    "зима": "пуховик",
    "весна": "пальто",
    "лето": "ветровка",
    "осень": "куртка",
}
​
​
if __name__ == '__main__':
    today = input("Какое сейчас время года? ")
    print_value(today, seasons)

Выполним этот файл еще раз и убедимся, что он работает:

А теперь запустим файл restraunt.py и убедимся, что исполняемый код из weather.py в него не импортировался:

Для верности добавим наше условие и в файл restraunt.py - на случай, если ты захочешь импортировать что-то из него.

restraunt.py
import weather
​
dishes = {
    "понедельник": "суп",
    "вторник": "стейк",
    "среда": "паста",
    "четверг": "отбивная",
    "пятница": "фрикадельки",
}
​
if __name__ == '__main__':
    today = input("Какой сегодня день? ")
    weather.print_value(today, dishes)

Теперь все на своих местах! Теперь если ты захочешь написать скрипт, не забудь добавить условие if __name__ == '__main__' для исполняемой части. Благодаря этому, ты сможешь имортировать код (например, какую-то функцию) из этого файла в другой. Использование этой конструкции считается хорошим тоном, так что старайся использовать ее каждый раз, как пишешь скрипт.
